## 核心设计原则

1. **合规底线**：不植入虚假记忆、不直接"下载知识"、仅做引导/体验/记忆强化/巩固
2. **技术可行**：全部基于现有非侵入式BCI、睡眠科学、多模态诱导技术
3. **工程化**：结构清晰、可直接落地开发
4. **隐私优先**：数据本地加密、用户授权管理

## 系统核心参数

| 参数 | 值 |
|------|-----|
| 系统名称 | 造梦府 DreamLearn |
| 核心技术 | 非侵入式BCI（EEG/EOG/ECG）+ REM睡眠监测 + 多模态诱导 |
| 预期效率提升 | 睡眠中学习效率提升 5-7 倍 |
| 目标用户 | 18岁以上成年人（首批）、K12学生（合规版本） |
| 合规认证目标 | 中国NMPA Class II、医疗器械注册 |

## 四层架构概览

```
┌─────────────────────────────────────────────────────────────┐
│  内容层（Content Layer）                                      │
│  历史沉浸 / 语言情境 / 考点速记 / 技能流程                      │
├─────────────────────────────────────────────────────────────┤
│  交互/诱导层（Interaction Layer）                             │
│  弱光模块 / 声学模块 / 振动模块 / 香氛模块                      │
├─────────────────────────────────────────────────────────────┤
│  计算层（Computation Layer）                                  │
│  边缘AI推理引擎 + 云端大模型集群                               │
├─────────────────────────────────────────────────────────────┤
│  感知层（Perception Layer）                                   │
│  EEG采集 + EOG眼动 + ECG心电 + 睡眠分期算法                    │
└─────────────────────────────────────────────────────────────┘
```

## 核心功能模块索引

| 模块编号 | 模块名称 | 核心功能 |
|----------|----------|----------|
| M01 | 睡眠监测与REM识别 | 实时睡眠分期、REM窗口预测 |
| M02 | 清醒梦诱导引擎 | MILD/SWILD诱导方案执行 |
| M03 | 内容神经编码编译器 | 学习内容→梦境信号转换 |
| M04 | 个性化适配AI | 用户模型迭代、内容难度调整 |
| M05 | 醒后复盘系统 | 学习报告生成、记忆巩固卡片 |
| M06 | 未成年人安全模式 | 内容过滤、使用时长管控 |
| M07 | 数据安全模块 | 本地加密、隐私保护 |
| M08 | 实验数据面板 | 研究者数据可视化 |

   │           │           │           │           │           │
   ▼           ▼           ▼           ▼           ▼           ▼
